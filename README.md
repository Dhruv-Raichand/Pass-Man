## ğŸš€ Getting Started

### Prerequisites

-   Node.js version 18 or higher
-   npm or yarn

### Installation

Clone the repository  
git clone [https://github.com/Dhruv-Raichand/Pass-Man.git](https://github.com/Dhruv-Raichand/Pass-Man.git)

Navigate into the project  
cd Pass-Man

Install dependencies  
npm install

### Run the App

npm run dev

Open your browser at  
[http://localhost:5173](http://localhost:5173)
